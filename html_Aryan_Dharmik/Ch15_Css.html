🧾 What is CSS? CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It controls how elements look on the screen, such as: Layout Colors Fonts Spacing Animations Why Use CSS? Without CSS, all websites would look like plain black text on a white background. CSS gives you control over: Visual design (colors, fonts, backgrounds) Layout (grid, flexbox, position, display) Responsive design (media queries) User interaction (hover, transitions) selector { property: value; } h1 { color: blue; font-size: 30px; } Types of CSS Inline CSS (inside an HTML tag)

CSS3 is the latest evolution of CSS with powerful new modules and features. It makes web design easier, more flexible, and modern. ✅ Key Additions in CSS3: Feature Description 🎨 Rounded corners border-radius 🖼️ Gradients linear-gradient, radial-gradient ✨ Box Shadows box-shadow 🌈 Text Effects text-shadow, custom fonts via @font-face 📏 Media Queries Responsive design 🧱 Flexbox & Grid Modern layout systems 🎞️ Animations & Transitions Smooth effects and motion 🧬 Transformations rotate, scale, translate, skew 🧩 Pseudo-classes :hover, :nth-child, etc. 🧪 Examples of CSS3 Features 1. Rounded Corners .box { border-radius: 10px; } 2. Box Shadow .box { box-shadow: 2px 2px 10px rgba(0, 0, 0, 0.3); } 3.button { transition: background 0.3s ease; } button:hover { background: green; } 4. Media Query (Responsive Design) @media (max-width: 600px) { body { background-color: lightgray; } } 5. Flexbox .container { display: flex; justify-content: center; align-items: center; } 🔄 Difference Between CSS and CSS3 Feature CSS (Old) CSS3 (New) Layout float, position Flexbox, Grid Animation Not supported Yes (transition, keyframes) Media Queries No Yes Shadows No Yes Custom Fonts Limited Yes (@font-face) Modules Monolithic Modular (Box Model, Text, etc.) 🧠 Conclusion CSS is essential for styling web pages. CSS3 introduced modern, flexible, and dynamic styling features. Using CSS3 makes sites responsive, interactive, and professional-looking.
