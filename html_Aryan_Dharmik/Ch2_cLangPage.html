C Language Topics

  1. Datatypes :

    In C, there are four types of datatypes. 1. Integer 2. Char 3. Bool 4. Float/Double

  2. Array :

    Array is collection of similar data items. e.g. int arr[10] can store 10 integer numbers. All these numbers are stored sequentially in continous memory locations.

  3. String :

    In C, strings are represented as arrays of characters, terminated by a null character \0. Unlike some other languages, C doesn't have a built-in string type. .

  4. Functions :

    In C, a function is a block of code that performs a specific task. Functions are essential for structuring programs, promoting reusability, and improving code readability.

  5. Pointers :

    Pointers in C are variables that store memory addresses. Instead of holding a direct value, a pointer contains the location in memory where a value is stored. This allows direct manipulation of data at specific memory locations.
